First-order logic and completeness

The grammar mathematics writes itself in: predicates, quantifiers, structures, and the bridge between syntactic proofs and semantic truth that Gödel built. By the end you'll see why compactness gives non-standard reals, why ZFC has a countable model, and why algebraically closed fields are tame.

Naive set theory gives us collections to talk about; first-order logic gives us a language to talk with. A structure picks specific sets and operations to play the parts; a theory is a list of axioms; a model of the theory is any structure satisfying every axiom. Gödel's completeness theorem says: a sentence is provable from $T$ iff it is true in every model of $T$. From there flow compactness (and non-standard analysis), Löwenheim–Skolem (and Skolem's paradox), and a clean theory of which theories are decidable.

Plan. Sections 1–3 build the apparatus — syntax, semantics, deduction. Section 4 proves Gödel's completeness theorem (Henkin's argument). Sections 5–7 unpack the consequences: compactness produces non-standard models of arithmetic, Löwenheim–Skolem produces countable models of ZFC, and quantifier elimination identifies tame theories like ACF and DLO.

1. Syntax: languages, terms, formulas

A first-order language $\mathcal{L}$ is a triple of disjoint symbol sets, each labelled with an arity:

  • Constants $c_1, c_2, \ldots$ (arity $0$ — name a single element).
  • Function symbols $f, g, \ldots$ (arity $\ge 1$ — take that many inputs, return one output).
  • Relation symbols $R, P, \ldots$ (arity $\ge 1$ — take inputs, return true/false).

To these we always add a fixed countable supply of variables $x, y, z, \ldots$, the equality symbol $=$, the connectives $\neg, \land, \lor, \to, \leftrightarrow$, the quantifiers $\forall, \exists$, and parentheses. Terms are built from variables and constants by applying function symbols; formulas are built from atomic relations $R(t_1,\ldots,t_n)$ and equalities $t_1=t_2$ by Boolean connectives and quantifiers. A sentence is a formula with no free variables.

Two languages worth keeping in mind.
  • Language of arithmetic: $\mathcal{L}_{\mathrm{ar}} = \{0, 1, +, \cdot, <\}$ — constants $0, 1$; binary functions $+, \cdot$; binary relation $<$.
  • Language of groups: $\mathcal{L}_{\mathrm{gp}} = \{e, \cdot, ^{-1}\}$ — constant $e$; binary $\cdot$; unary $^{-1}$.
The choice of language is a design decision: it fixes what counts as an atomic statement.

The trickiest move at this level is keeping track of free versus bound variables. In $\forall x\,(P(x) \to Q(x,y))$ the variable $x$ is bound by the quantifier, while $y$ remains free. Only sentences (closed formulas) can be true or false in a structure — open formulas need an assignment of values to free variables before truth makes sense.

2. Tarski semantics and structures

Syntax is just symbols on paper; semantics assigns meaning. An $\mathcal{L}$-structure $\mathfrak{A}$ consists of:

  • A non-empty domain (or universe) $A$.
  • For each constant symbol $c$, an element $c^{\mathfrak{A}} \in A$.
  • For each $n$-ary function symbol $f$, a function $f^{\mathfrak{A}} \colon A^n \to A$.
  • For each $n$-ary relation symbol $R$, a subset $R^{\mathfrak{A}} \subseteq A^n$.

Given an assignment $s\colon \mathrm{Var}\to A$, every term $t$ denotes an element $t^{\mathfrak{A}}[s]\in A$ (by recursion on term structure). Then the satisfaction relation $\mathfrak{A}\models\varphi[s]$ is defined recursively on formulas:

formulasatisfied iff
$t_1 = t_2$$t_1^{\mathfrak{A}}[s] = t_2^{\mathfrak{A}}[s]$
$R(t_1,\ldots,t_n)$$(t_1^{\mathfrak{A}}[s], \ldots, t_n^{\mathfrak{A}}[s]) \in R^{\mathfrak{A}}$
$\neg\varphi$$\mathfrak{A}\not\models\varphi[s]$
$\varphi\land\psi$$\mathfrak{A}\models\varphi[s]$ and $\mathfrak{A}\models\psi[s]$
$\forall x\,\varphi$$\mathfrak{A}\models\varphi[s[x\mapsto a]]$ for every $a\in A$
$\exists x\,\varphi$$\mathfrak{A}\models\varphi[s[x\mapsto a]]$ for some $a\in A$

For a sentence $\varphi$ (no free variables) the assignment is irrelevant, and we just write $\mathfrak{A}\models\varphi$. A theory $T$ is a set of sentences. We say $\mathfrak{A}\models T$ iff $\mathfrak{A}$ models every sentence in $T$. Two derived notions:

Satisfiability and entailment. A theory $T$ is satisfiable (or consistent, semantically) iff some structure models it. We write $T\models\varphi$ to mean every model of $T$ satisfies $\varphi$.

3. Deduction: proof systems and soundness

So far we've described what is true in a structure. A complementary question: what is provable from a list of axioms? We fix some proof system — a list of logical axioms (tautologies of pure logic) and inference rules (e.g. modus ponens). A formal proof from $\Gamma$ of $\varphi$ is a finite sequence of formulas, each either an axiom, in $\Gamma$, or obtained from earlier lines by an inference rule, ending in $\varphi$. We write $\Gamma\vdash\varphi$.

One Hilbert-style calculus. Take as logical axioms (schemes): \[ (\mathrm{K})\;\;\varphi\to(\psi\to\varphi),\qquad (\mathrm{S})\;\;(\varphi\to(\psi\to\chi))\to((\varphi\to\psi)\to(\varphi\to\chi)), \] \[ (\mathrm{N})\;\;(\neg\psi\to\neg\varphi)\to(\varphi\to\psi),\qquad (\forall\mathrm{I})\;\;\forall x\,\varphi\to\varphi[x:=t]\;(t\text{ free for }x), \] plus the inference rules modus ponens ($\varphi,\,\varphi\to\psi \;\Rightarrow\; \psi$) and generalisation ($\varphi \;\Rightarrow\; \forall x\,\varphi$, with the side condition that $x$ is not free in any premise of $\Gamma$). Equality has its own axioms ($t=t$ and substitution). This is barebones but complete.

The key meta-theorems linking proofs to truth:

Soundness. If $\Gamma\vdash\varphi$ then $\Gamma\models\varphi$. Equivalently, every theorem is valid in every model of the premises.

Soundness is proved by induction on proof length. The base case checks each axiom is valid in every structure; the inductive step checks each rule preserves validity (modus ponens because of the truth tables for $\to$; generalisation because of its side condition). The hard direction — completeness, the converse implication — is §4.

One useful fact provable at this stage:

Deduction theorem. $\Gamma \cup \{\varphi\} \vdash \psi$ iff $\Gamma \vdash \varphi \to \psi$ (provided no application of generalisation in the proof on the left binds a variable free in $\varphi$). It collapses the gap between "assume $\varphi$, derive $\psi$" and "prove $\varphi\to\psi$."

4. Gödel's completeness theorem

The headline theorem of first-order logic. It says that the rules of pure logic are just enough: every semantic consequence has a syntactic proof.

Completeness theorem (Gödel 1929). For any first-order theory $\Gamma$ and any sentence $\varphi$, $$\Gamma \vdash \varphi \;\;\Longleftrightarrow\;\; \Gamma \models \varphi.$$ Equivalently: a theory $T$ is consistent (no proof of $\bot$) iff $T$ has a model.

One direction ($\Rightarrow$) is soundness, proved in §3. The substantive direction ($\Leftarrow$) is what people mean by "the completeness theorem." We sketch Henkin's proof (1949), the version every modern textbook uses.

Henkin's construction

Suppose $T$ is a consistent theory in language $\mathcal{L}$. We will manufacture a model directly out of syntax. Three moves:

  1. Add witnesses. For every existential formula $\exists x\,\psi(x)$ in $\mathcal{L}$, introduce a fresh constant $c_{\psi}$ and add the axiom $\exists x\,\psi(x) \to \psi(c_{\psi})$. The expanded language is $\mathcal{L}^*$ and the expanded theory $T^*$ is still consistent (proof: any proof of $\bot$ in $T^*$ uses only finitely many witnesses, and they can be discharged one at a time).
  2. Lindenbaum-extend to maximal consistency. Enumerate all $\mathcal{L}^*$-sentences $\varphi_0, \varphi_1, \ldots$. Set $T_0 = T^*$; at stage $n$, let $T_{n+1} = T_n \cup \{\varphi_n\}$ if that's consistent, else $T_n \cup \{\neg\varphi_n\}$. The union $T^{\mathrm{H}}$ is consistent, contains $T$, and decides every sentence — for every $\varphi$, exactly one of $\varphi, \neg\varphi$ is in $T^{\mathrm{H}}$.
  3. Build the term model. Let $C$ be the set of closed $\mathcal{L}^*$-terms. Define $s\sim t$ iff $T^{\mathrm{H}}\vdash s=t$; this is an equivalence relation by the equality axioms. Let $A = C/{\sim}$ be the domain. Interpret each function symbol by its syntactic action on representatives, each relation symbol by $T^{\mathrm{H}}$'s decision. The resulting structure $\mathfrak{A}$ satisfies the Truth Lemma: $\mathfrak{A}\models\varphi$ iff $\varphi\in T^{\mathrm{H}}$. In particular $\mathfrak{A}\models T$.

The Truth Lemma is the keystone, and the witness step is what makes existential cases work: $\mathfrak{A}\models\exists x\,\psi(x)$ iff $T^{\mathrm{H}}$ contains $\exists x\,\psi(x)$, iff $T^{\mathrm{H}}\vdash\psi(c_{\psi})$, iff $[c_{\psi}]\in A$ witnesses $\psi$.

Why this matters. Completeness reduces "is $\varphi$ a logical consequence of $T$?" to "can I find a finite proof?" — a recursively enumerable question. It is the foundation of model theory and of automated theorem proving. It also implies compactness directly (next section), which has surprising mathematical consequences.

5. Compactness and non-standard models

The first big consequence of completeness is so important it gets its own name. It says satisfiability is a "local" condition: if every finite portion of a theory is satisfiable, the whole theory is.

Compactness theorem. A set of first-order sentences $\Sigma$ is satisfiable iff every finite subset of $\Sigma$ is satisfiable.

Proof: $\Sigma$ is satisfiable iff (by completeness) consistent iff (definitionally) no proof of $\bot$ exists. A proof is finite, so it uses only finitely many sentences from $\Sigma$. Therefore $\Sigma\vdash\bot$ iff some finite $\Sigma_0\subseteq\Sigma$ has $\Sigma_0\vdash\bot$. Negate.

The applications are wonderful and somewhat surprising. Here is the prototype:

Non-standard models of arithmetic. Let $\mathrm{Th}(\mathbb{N})$ be the (massive) set of all sentences true in the standard naturals. Add a fresh constant $c$, and consider $$\Sigma \;=\; \mathrm{Th}(\mathbb{N}) \;\cup\; \{c \neq \overline{0},\; c\neq\overline{1},\; c\neq\overline{2}, \ldots\}$$ where $\overline{n}$ is the numeral $1+1+\cdots+1$ ($n$ times). Every finite subset is satisfied by interpreting $c$ as a sufficiently large standard natural. By compactness, $\Sigma$ has a model — a structure $\mathfrak{A}$ satisfying every truth of arithmetic, with an element $c^{\mathfrak{A}}$ that is not equal to any standard numeral. That element is "infinite" in $\mathfrak{A}$, and $\mathfrak{A}$ is a non-standard model of arithmetic.

By the same recipe applied to $\mathbb{R}$ in the language of ordered fields, one builds the hyperreal numbers, an extension of $\mathbb{R}$ containing infinitesimals — the foundation of non-standard analysis.

Other compactness corollaries.
  • If a theory has arbitrarily large finite models, it has an infinite model. (Add the axioms "there are at least $n$ distinct elements" for every $n$.)
  • If $\varphi$ holds in every algebraically closed field of characteristic $0$, it holds in every algebraically closed field of sufficiently large prime characteristic. (Used constantly in algebraic geometry as the "transfer principle.")
  • The class of well-orderings cannot be axiomatised in first-order logic. (If it could, compactness would give an ill-ordered model — contradiction.)

6. Löwenheim–Skolem and Skolem's paradox

Compactness says first-order logic cannot pin down infinity from above. The Löwenheim–Skolem theorems say it cannot pin down infinity from below either.

Downward Löwenheim–Skolem. If a theory $T$ in a countable first-order language is satisfiable, then $T$ has a countable model. More generally, if $T$ has an infinite model and $|T|+\aleph_0\le\kappa$, then $T$ has a model of cardinality $\kappa$.
Upward Löwenheim–Skolem. If $T$ has an infinite model, then for every cardinal $\kappa\ge|T|+\aleph_0$, $T$ has a model of cardinality $\kappa$.

Together: an infinite first-order theory in a countable language has models of every infinite cardinality. The cardinality of a model is invisible to first-order axioms.

Skolem's paradox

Apply the downward theorem to ZFC: ZFC is a countable theory in a countable language. If ZFC is consistent (which we believe), then by completeness it has a model; by downward LS, it has a countable model $M$. But ZFC proves "$\mathbb{R}$ is uncountable." So inside $M$, there is an element $\mathbb{R}^M$ which $M$ believes to be uncountable, even though externally $|\mathbb{R}^M|\le|M|=\aleph_0$.

Resolution. "Uncountable in $M$" means: there is no element of $M$ that is a bijection $\mathbb{N}^M\to\mathbb{R}^M$. The external bijection between $\mathbb{R}^M$ and a subset of $\mathbb{N}$ exists in our metatheory but is not an element of $M$. Cardinality is not absolute between models — it is a second-order property that first-order logic cannot fully express.

The lesson is sharp: first-order logic captures the relations between elements a structure declares, not its overall size. If you want to talk about "the" reals, you need second-order logic or a fixed background model.

7. Decidable theories and quantifier elimination

Completeness gives an algorithm idea: to test whether $T\vdash\varphi$, enumerate all proofs from $T$ in some fixed system. If $T$ is complete (decides every sentence) and recursively axiomatised (the axioms are an algorithmically enumerable set), this enumeration eventually finds a proof of $\varphi$ or of $\neg\varphi$, and we know which.

Decidability. A theory $T$ is decidable iff there is an algorithm that, given any sentence $\varphi$, returns whether $T\vdash\varphi$. Theorem: a complete, recursively axiomatised first-order theory is decidable.

The catch is completeness in the sense of "decides every sentence" — distinct from Gödel's completeness theorem above. Many natural theories are incomplete (PA, ZFC, the theory of groups), so they don't get decidability for free. But several important ones are complete in the model-theoretic sense, and they admit a powerful structural feature called quantifier elimination.

Quantifier elimination (QE). A theory $T$ admits quantifier elimination in $\mathcal{L}$ iff every $\mathcal{L}$-formula $\varphi(\bar x)$ is $T$-equivalent to a quantifier-free formula $\psi(\bar x)$ in the same free variables. Algorithm: replace bound quantifiers by polynomial Boolean conditions, layer by layer.

Theories with QE in their natural language:

theorylanguagetypical formula post-QE
DLO (dense linear orders without endpoints)$\{<\}$Boolean combinations of $x_i < x_j$
ACF$_p$ (algebraically closed fields, char $p$)$\{0,1,+,\cdot\}$Boolean combinations of $f(\bar x)=0$, $f$ a polynomial
RCF (real closed fields, e.g. $\mathbb{R}$)$\{0,1,+,\cdot,<\}$Boolean combinations of $f(\bar x)\ge 0$ (Tarski–Seidenberg)
$(\mathbb{Q}_p, +, \cdot)$ ($p$-adics)language of valued fieldspolynomial inequalities, divisibility (Macintyre)

QE has consequences far beyond decidability. For ACF, it gives Chevalley's theorem: the image of a constructible set under a polynomial map is constructible (since "$\exists y$" of a quantifier-free condition becomes another quantifier-free condition). For RCF, it gives semi-algebraic geometry: projections of semi-algebraic sets are semi-algebraic. These are foundational facts of algebraic geometry and real algebraic geometry, both consequences of pure model theory.

Counter-example. True arithmetic $\mathrm{Th}(\mathbb{N},+,\cdot,0,1)$ does not admit QE in this language and is in fact undecidable (Tarski). Multiplication interacting with addition allows enough self-reference to encode Turing computation; Gödel's incompleteness theorem and Tarski's undefinability theorem block any algorithm that decides truth.

8. Connections

First-order logic is the substrate every other formal mathematics in this notebook is built on top of. The theorems above are the moves it gives you; the bridges below are where the moves cash out.

Set theory and Foundations

ZFC is a first-order theory in the language $\{\in\}$, and every metatheorem about FOL transfers — ZFC and ordinals are interpretations of the abstract machinery here. Compactness gives non-standard models of arithmetic; Löwenheim–Skolem produces countable models of set theory and exposes Skolem's paradox.

Model theory and stability

Model theory takes the semantic side and runs with it: types, saturation, Ehrenfeucht–Fraïssé, and stability theory all live one layer above completeness.

Computability and incompleteness

Gödel's first incompleteness theorem follows from compactness + arithmetisation; computability develops the recursive-function side. Decidable theories (presburger, dense linear orders) live one layer below.

Forcing and independence

When ZFC is too weak to decide a sentence (CH, Suslin), the meta-tool is forcing — a generic-extension construction interpretable as choosing a particular model of the original theory.

Open frontiers: nonstandard analysis (Robinson) and ultrafilter constructions; descriptive set theory and effective Borel reducibility; reverse mathematics — exactly which axioms of arithmetic / set theory are required to prove which classical theorems.
