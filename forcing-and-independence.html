Forcing and independence

Cohen's 1963 technique for building a model of ZFC where you decide what is true. Pile up partial conditions until a generic filter exists; the conditions force statements into the extension; choose the poset to make the continuum hypothesis fail. The same machine settles Suslin, Whitehead, and a long list of other questions ZFC alone refuses to answer.

Gödel had shown in 1940 that CH is consistent with ZFC, by building the constructible universe $L$. The other direction — is $\neg$CH also consistent? — sat open for two decades. Cohen's forcing answered yes, and along the way invented the most flexible tool in modern set theory: a way to construct, from a model $M$ of ZFC, a larger model $M[G]$ in which a designed list of new sets exists.

The plot of this page: assemble a poset $\mathbb P$ of "partial information" called conditions; identify $\mathbb P$-generic filters $G$ over a countable model $M$; show that the extension $M[G]$ is again a model of ZFC and that statements about $G$ are decided by a syntactically defined forcing relation $p\Vdash\varphi$. Pick $\mathbb P=2^{<\omega}$ to add a new real, or its $\aleph_2$-fold product to add $\aleph_2$ reals at once and make CH fail. Then survey other independence theorems and Martin's axiom.

1. Forcing posets and dense subsets

A forcing notion is a partially ordered set $(\mathbb P,\le)$ with a maximum element $\mathbb 1\in\mathbb P$. Read $p\le q$ as "$p$ is stronger than $q$" — $p$ commits to more about the unknown object you are building.

Sign conventions vary. Cohen and Kunen use $p\le q$ for "$p$ is stronger" (going down the order pins more bits). Some authors flip the order. We use Cohen/Kunen here. The maximum $\mathbb 1$ is the empty commitment — "I have decided nothing yet".

The canonical example: Cohen forcing $\mathbb P=2^{<\omega}$, the set of finite binary strings, ordered by extension. The empty string $\langle\rangle=\mathbb 1$ is on top; $\langle 0\rangle$ and $\langle 1\rangle$ are stronger; each condition has exactly two immediate extensions. The poset is the infinite binary tree growing downward.

A subset $D\subseteq\mathbb P$ is dense iff for every $p\in\mathbb P$ there exists $q\in D$ with $q\le p$. Density says: every condition can be extended into $D$. Each dense set is a question we want answered, and meeting it is an answer.

Two conditions $p,q$ are compatible if some $r\le p,q$ exists; otherwise incompatible (notation $p\perp q$). In Cohen forcing, two strings are compatible iff one extends the other. An antichain is a set of pairwise incompatible conditions; the countable chain condition (ccc) says every antichain is countable. Cohen forcing satisfies ccc — it preserves all cardinals, a key technical fact for the $\neg$CH application below.

2. Generic filters and the model $M[G]$

Fix a countable transitive model $M\models$ ZFC and a poset $\mathbb P\in M$. A subset $G\subseteq\mathbb P$ is a filter if it is upward-closed ($p\in G,\ q\ge p\Rightarrow q\in G$) and downward-directed ($p,q\in G\Rightarrow\exists r\in G,\ r\le p,q$). It is $M$-generic if additionally:

\[ G\cap D \ne \emptyset \quad\text{for every dense } D\in M. \]

The countability of $M$ is essential: there are only countably many dense sets in $M$, so we can enumerate them $D_0,D_1,\dots$ and pick a descending chain $p_0\ge p_1\ge\dots$ with $p_n\in D_n$. The upward closure of this chain is the desired generic filter — and it lives outside $M$, since meeting every dense set is precisely what no $M$-element does.

From a generic $G$ we construct $M[G]$ by interpreting $\mathbb P$-names from $M$. A name $\dot x\in M$ is a set of pairs $(\dot y, p)$; its evaluation is $\dot x_G = \{\,\dot y_G : (\dot y, p)\in\dot x \text{ for some } p\in G\,\}$. The model is $M[G] = \{\dot x_G : \dot x\in M\}$.

Three theorems (Cohen). For any countable transitive $M\models$ ZFC, any $\mathbb P\in M$, and any $M$-generic $G\subseteq\mathbb P$:
  • Existence of $G$. $M$-generic filters always exist (in $V$, outside $M$).
  • $M\subseteq M[G]$ and $G\in M[G]$. The extension keeps everything from the ground model and exhibits $G$ as a set.
  • $M[G]\models$ ZFC. The new model satisfies all the same axioms.

So $M[G]$ is a legitimate place to do mathematics. The question — what new statements hold there — is settled by the forcing relation in the next section.

3. The forcing relation and truth lemma

So far the construction has been semantic: pick $G$, look at $M[G]$. The miracle is that everything true in $M[G]$ is decided by data already in $M$. The bookkeeping device is the forcing relation $p\Vdash\varphi$, defined inside $M$ by recursion on the formula $\varphi$ (which is built from $\mathbb P$-names). Read $p\Vdash\varphi$ as "every generic $G$ containing $p$ makes $\varphi$ true in $M[G]$".

Forcing relation — three properties.
  • Definability. The relation $\Vdash$ is definable in $M$. So $M$ knows which conditions force which formulas.
  • Monotonicity. If $p\Vdash\varphi$ and $q\le p$, then $q\Vdash\varphi$. Stronger conditions inherit forced statements.
  • Density. For every $\varphi$, the set $\{p : p\Vdash\varphi \text{ or } p\Vdash\neg\varphi\}$ is dense in $\mathbb P$. So every generic decides every formula.
Truth lemma. For every formula $\varphi(\dot x_1,\dots,\dot x_n)$ in the forcing language and every $M$-generic $G$: \[ M[G]\models\varphi(\dot x_{1,G},\dots,\dot x_{n,G}) \iff \exists p\in G,\ p\Vdash\varphi(\dot x_1,\dots,\dot x_n). \] Equivalently: truth in $M[G]$ is decided by the conditions that ended up in $G$.

4. Cohen forcing for adding reals

The poset $\mathbb P=2^{<\omega}$ — the binary tree of finite strings — is Cohen forcing. A generic filter $G$ is a chain of finite strings cofinal in length, and its collapse is a single new infinite binary sequence:

\[ r_G \;=\; \bigcup_{p\in G} p \;\in\; 2^\omega. \]

The Cohen real $r_G$ is genuinely new: $r_G\notin M$. To see it, fix any $s\in (2^\omega)^M$. The set $D_s=\{\,p\in 2^{<\omega} : \exists n,\ p(n)\ne s(n)\,\}$ is in $M$ and is dense: given $p$, extend $p$ by appending the bit opposite to $s$ at the next index. Genericity of $G$ then forces $r_G\ne s$. Since this argument runs for every $s\in M$, the Cohen real avoids every old real.

Side-by-side product: forcing $\neg$CH

To add many new reals at once, take instead

\[ \mathbb P \;=\; \operatorname{Fn}(\aleph_2\times\omega,\, 2), \]

the set of finite partial functions from $\aleph_2\times\omega$ to $\{0,1\}$, ordered by extension. A condition is a finite list of commitments "at index $\alpha\in\aleph_2$ and bit position $n\in\omega$, the value is $b$". A generic $G$ defines $\aleph_2$ many distinct reals $r_\alpha\in 2^\omega$ in $M[G]$, indexed by $\alpha<\aleph_2^M$.

Cardinal preservation. $\mathbb P$ has the countable chain condition (ccc): every antichain is countable, by a $\Delta$-system argument on the finite supports. The ccc preserves cardinals and cofinalities — no $\aleph_\alpha^M$ collapses in $M[G]$. So $\aleph_2^{M[G]}=\aleph_2^M$, and the new reals stay distinct as elements of $2^\omega$ in $M[G]$.

Conclusion: $|2^\omega|^{M[G]}\ge\aleph_2^{M[G]} > \aleph_1^{M[G]}$, so $M[G]\models\neg\mathrm{CH}$. We will draw the picture in the next section.

5. Independence of the continuum hypothesis

Putting the pieces together. Let $M$ be a countable transitive model of ZFC + CH (Gödel's $L$ if you like). Let $\mathbb P=\operatorname{Fn}(\aleph_2^M\times\omega,2)$. Let $G$ be $M$-generic. Then:

stepfacttool
1$M[G]\models$ ZFCgeneric extension theorem
2$\aleph_2^{M[G]}=\aleph_2^M$ (no collapse)ccc + $\Delta$-system lemma
3$M[G]$ contains $\aleph_2^M$ pairwise distinct realsdensity: for $\alpha\ne\beta$, $D_{\alpha\beta}=\{p:p$ distinguishes $r_\alpha,r_\beta\}$ is dense
4so $|2^\omega|^{M[G]}\ge\aleph_2^{M[G]} > \aleph_1^{M[G]}$by 2 + 3
5$M[G]\models\neg\mathrm{CH}$by 4
Independence of CH. Combined with Gödel's $L\models$ ZFC + CH:
  • ZFC + CH is consistent (Gödel 1940, $L$).
  • ZFC + $\neg$CH is consistent (Cohen 1963, forcing).
So neither CH nor $\neg$CH is provable in ZFC: CH is independent of ZFC.

6. Other independence results and Martin's axiom

Forcing did not stop with CH. The same general machine — pick a poset, force, decode the truth in the extension via $\Vdash$ — settles a long list of questions ZFC cannot.

Martin's axiom — a parametric weakening of CH

Martin's axiom MA says: for every ccc poset $\mathbb P$ and every family $\mathcal D$ of $<2^{\aleph_0}$ dense sets, a filter $G\subseteq\mathbb P$ meeting every $D\in\mathcal D$ exists. Under CH, MA is automatic (only countable families). Under $\neg$CH, it is a substantive axiom that asserts generic-like filters exist for small problems even though no truly generic filter does.

MA + $\neg$CH is consistent (Solovay–Tennenbaum). It implies the Suslin hypothesis, that $2^{\aleph_0}$ is regular, that every Aronszajn tree is special, and many combinatorial principles. Working set theorists routinely treat MA as an "extra dial" sitting alongside ZFC.

The big picture. Independence is the rule, not the exception, once you leave the safe haven of arithmetic. The set-theoretic universe is not a single object — it is a multiverse, and forcing is the device that lets us hop between members. CH is the most famous member of the long list above, but everyday questions in algebraic topology (Whitehead), measure theory (cardinal invariants of the continuum), and analysis (the Suslin problem) all live there too.

7. Connections

Forcing is one of the few set-theoretic techniques that has reshaped how mathematicians think about provability — its consequences spill out of pure set theory into model theory, topology, and combinatorics.

Continuum hypothesis and beyond

Cohen's theorem closes the question Hilbert posed in 1900 — but only by saying both CH and $\neg$CH are unfalsifiable. CH inside ZFC develops the upside; model theory uses the same machinery to construct saturated structures of any cardinality.

Gödel's constructible universe $L$

Forcing extends downward via inner models: $L$ is a model of ZFC + GCH + V=L, and Cohen extensions are the orthogonal direction. First-order logic supplies the syntactic backbone every inner-model argument runs on.

Topological forcing and Boolean-valued models

A forcing poset $\mathbb{P}$ is the same data as a complete Boolean algebra of regular open sets in a topology. Topology and topos theory are the natural conceptual frames; sheaf-theoretic forcing is the modern recasting.

Combinatorics of the continuum

Independence results for cardinal invariants of the continuum (Suslin's problem, Whitehead's problem, Borel-conjecture failures) read as a forcing-driven inventory of which combinatorial questions are decided by ZFC and which are not.

Open frontiers: large-cardinal axioms (measurable, Woodin, supercompact) which provide a "natural" indestructibility theory for forcing extensions; iterated forcing and proper-forcing axioms (PFA, MM); inner-model program (Mitchell, Steel) attempting canonical models for very large cardinals.
