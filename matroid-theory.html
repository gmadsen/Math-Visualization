Matroid theory

A matroid is the combinatorial skeleton shared by linear independence and graph acyclicity. The data is sparse — a finite ground set $E$ together with a family of "independent" subsets — yet it is enough to make the greedy algorithm provably optimal, to define a duality that swaps every theorem with its partner, and to encode a polynomial whose specializations include the chromatic polynomial, network reliability, and the partition function of a Potts model. This page traces the cryptomorphic axiom systems and their interplay.

1. Independence axioms

Whitney's 1935 starting point. Fix a finite ground set $E$. A matroid on $E$ is a non-empty family $\mathcal{I} \subseteq 2^E$ of subsets called independent sets, satisfying:

  • (I1) Non-triviality. $\emptyset \in \mathcal{I}$.
  • (I2) Hereditary. If $I \in \mathcal{I}$ and $J \subseteq I$, then $J \in \mathcal{I}$.
  • (I3) Exchange. If $I, J \in \mathcal{I}$ with $|I| < |J|$, there exists $e \in J \setminus I$ such that $I \cup \{e\} \in \mathcal{I}$.

(I1) and (I2) alone define a simplicial complex — a downward-closed family. The novel content is (I3), the augmentation or exchange axiom: any smaller independent set can always be enlarged from any strictly larger one. It forces every maximal independent set to have the same size; without it, "independent" becomes a meaningless label.

Two motivating examples

Both examples below satisfy (I1)–(I3); the abstract definition is engineered to capture the common skeleton.

  • Vector matroid $M[A]$. Let $A$ be an $r \times n$ matrix over a field $k$, with columns indexed by $E = \{1,\ldots,n\}$. Set $\mathcal{I} = \{S \subseteq E : \text{columns}\ \{a_i : i \in S\}\ \text{are linearly independent}\}$. Hereditary is obvious; exchange is the Steinitz exchange lemma from linear algebra.
  • Graphic matroid $M(G)$. Let $G$ be a finite graph with edge set $E$. Set $\mathcal{I} = \{F \subseteq E : F\ \text{contains no cycle}\}$ — the forests of $G$. Hereditary because removing edges cannot create a cycle. Exchange: if $F, F'$ are forests with $|F| < |F'|$, then $F$ has more connected components than $F'$, so some edge of $F'$ joins two components of $F$ and can be added without forming a cycle.
Why the same axioms? The Steinitz exchange and the forest exchange feel like different facts. The matroid abstraction says they are not — both are instances of (I3). The dictionary then propagates: a basis (max independent), a circuit (min dependent), a dual matroid, a rank function, etc., all have parallel meanings on both sides. The pages on field extensions and cycles in algebraic topology are downstream consumers of this skeleton.

2. Bases, rank, and cryptomorphism

A maximal independent set is a basis. The exchange axiom (I3) forces all bases to have the same cardinality, called the rank $r(M)$. More generally, for any $S \subseteq E$ define $$r(S) \;=\; \max\{\,|I| : I \subseteq S,\ I \in \mathcal{I}\,\}.$$ This rank function $r : 2^E \to \mathbb{Z}_{\ge 0}$ is the second face of a matroid.

Basis exchange

From (I3) one extracts the cleaner symmetric statement: for any two bases $B, B'$ and any element $x \in B \setminus B'$, there exists $y \in B' \setminus B$ with $(B \setminus \{x\}) \cup \{y\}$ also a basis. You "swap one in, one out." This basis-exchange axiom is itself enough to recover the whole matroid — the family of bases determines $\mathcal{I}$ as $\{I : I \subseteq B\ \text{for some basis}\ B\}$.

Rank-function axioms

The rank function is characterized by three properties:

  • (R1) $0 \le r(S) \le |S|$.
  • (R2) Monotone: $S \subseteq T \implies r(S) \le r(T)$.
  • (R3) Submodular: $r(S \cup T) + r(S \cap T) \le r(S) + r(T)$.

Conversely, any function $r$ satisfying (R1)–(R3) with integer values defines a matroid by $\mathcal{I} = \{S : r(S) = |S|\}$ — these are the sets where every element contributes one to the rank. The phenomenon that bases, independent sets, rank, closure, and circuits each give an equivalent axiomatization of the same object is called cryptomorphism; one matroid wears many costumes.

Submodularity is the heart. The inequality $r(S \cup T) + r(S \cap T) \le r(S) + r(T)$ is the combinatorial fingerprint that distinguishes matroids from arbitrary simplicial complexes. It says rank exhibits "diminishing returns": adding $T$ on top of $S$ gains less than adding $T$ to a smaller intersection. This is exactly why the greedy algorithm works (§ 6).

3. Vector, graphic, uniform, transversal

The class of matroids is wide enough to be interesting and narrow enough to be tractable. Four canonical families:

Uniform matroid $U_{r,n}$

Ground set $\{1,\ldots,n\}$; independent sets are all subsets of size $\le r$. Every $r$-subset is a basis; the rank function is $r(S) = \min(|S|, r)$. The case $U_{n,n}$ is "everything is independent" (the free matroid); $U_{0,n}$ is the trivial matroid. $U_{2,4}$ is the smallest non-representable example over $\mathbb{F}_2$ — it cannot arise as a vector matroid over the two-element field — and is a useful sanity-check counterexample.

Vector matroid $M[A]$

Already seen. A matroid is $k$-representable if it equals $M[A]$ for some matrix $A$ over the field $k$. Not every matroid is representable — the smallest non-representable example is the Vámos matroid on $8$ elements, which fails to be a vector matroid over any field. Tutte (1958) gave an excluded-minor characterization of $\mathbb{F}_2$-representable matroids: $M$ is binary $\iff$ $U_{2,4}$ is not a minor of $M$.

Graphic matroid $M(G)$

Independent sets = forests of $G$. Bases = spanning forests; if $G$ is connected, bases = spanning trees. The rank is $r(M(G)) = |V(G)| - c(G)$ where $c$ counts connected components. Graphic matroids are always $\mathbb{F}_2$-representable (and in fact $\mathbb{F}_p$-representable for every $p$): take the signed incidence matrix.

Transversal matroid

Given a finite collection $\mathcal{A} = (A_1, \ldots, A_m)$ of subsets of $E$, declare $S \subseteq E$ independent if there is an injective system of distinct representatives — an injection $\phi : S \to \{1,\ldots,m\}$ with $s \in A_{\phi(s)}$ for all $s \in S$. Hall's marriage theorem shows this is a matroid; bases are maximum partial transversals. Transversal matroids are exactly the matroids of certain bipartite-graph matchings, and form the bridge between matroid theory and combinatorial optimization.

4. Closure, flats, and the geometric lattice

For $S \subseteq E$, the closure $\operatorname{cl}(S)$ is $$\operatorname{cl}(S) \;=\; \{\,e \in E : r(S \cup \{e\}) = r(S)\,\}.$$ That is, $\operatorname{cl}(S)$ contains every element that contributes nothing extra to the rank — exactly the "linear span" intuition for the vector-matroid case. A subset $F \subseteq E$ is a flat (or closed set) if $F = \operatorname{cl}(F)$.

Closure axioms

The closure operator $\operatorname{cl}: 2^E \to 2^E$ satisfies:

  • (C1) Extensive: $S \subseteq \operatorname{cl}(S)$.
  • (C2) Monotone: $S \subseteq T \implies \operatorname{cl}(S) \subseteq \operatorname{cl}(T)$.
  • (C3) Idempotent: $\operatorname{cl}(\operatorname{cl}(S)) = \operatorname{cl}(S)$.
  • (C4) Mac Lane–Steinitz exchange: if $y \in \operatorname{cl}(S \cup \{x\}) \setminus \operatorname{cl}(S)$, then $x \in \operatorname{cl}(S \cup \{y\})$.

(C1)–(C3) make $\operatorname{cl}$ a closure operator on a poset; the matroid content is (C4), which says "if $y$ depended on $S \cup \{x\}$ but not on $S$, then $x$ depended on $S \cup \{y\}$" — the closure-side avatar of the exchange axiom. Any operator satisfying (C1)–(C4) on $2^E$ is the closure of a unique matroid.

The lattice of flats

Order flats by inclusion. The result is a geometric lattice: graded (by rank), atomistic (every flat is the join of rank-$1$ flats, which are the singletons or "points"), and semimodular ($r(F \vee G) + r(F \wedge G) \le r(F) + r(G)$). Birkhoff's theorem (1935) says geometric lattices and simple matroids — those without loops or parallel elements — are the same data.

Projective planes are matroids. Take the points of $\mathrm{PG}(2,q)$ as the ground set; declare a $3$-subset independent unless its three points are collinear. The result is a rank-$3$ matroid whose flats are: $\emptyset$, the points, the lines, and $E$. The geometric-lattice structure recovers the incidence axioms of the projective plane. 5. Matroid duality

Every matroid $M$ on ground set $E$ has a dual matroid $M^*$ on the same ground set, defined by $$\mathcal{B}(M^*) \;=\; \{\, E \setminus B : B \in \mathcal{B}(M)\,\}.$$ The bases of $M^*$ are precisely the complements of the bases of $M$. Verifying this satisfies the basis-exchange axiom is a one-line consequence of the original exchange — and the construction is involutive: $(M^*)^* = M$.

Dual rank

The rank functions are linked by $$r_{M^*}(S) \;=\; |S| - r(M) + r(E \setminus S).$$ In particular $r(M) + r(M^*) = |E|$. Loops of $M$ are coloops of $M^*$; circuits of $M^*$ are cocircuits of $M$, the minimal cuts that disconnect a basis.

The dual of a graphic matroid

Here is the punchline. If $G$ is a planar graph and $G^*$ is its planar dual (vertex per face, edge per shared boundary), then $$M(G)^* \;=\; M(G^*).$$ Spanning trees of $G$ are in bijection with spanning trees of $G^*$ via edge-complement. For non-planar $G$, $M(G)^*$ exists as a matroid but is no longer graphic — Whitney's planarity criterion (1932) says $G$ is planar iff $M(G)^*$ is graphic, with the dual graph providing a witness. The same matroid-theoretic statement subsumes Kuratowski's $K_5$/$K_{3,3}$ criterion.

Every theorem has a co-theorem. A statement about matroids automatically has a dual: replace "basis" by "complement of basis", "circuit" by "cocircuit", "rank" by the dual formula. The greedy algorithm § 6 has a dual interpretation as a "reverse-greedy" deletion procedure. Tutte polynomial § 7 is symmetric under $T_M(x,y) = T_{M^*}(y,x)$. This duality is the matroid-theoretic shadow of the planar-graph face/vertex duality and of the linear-algebra duality between row space and column space.

6. The greedy algorithm

Suppose each $e \in E$ has a non-negative weight $w(e)$, and we want a maximum-weight independent set. The greedy algorithm:

  1. Sort the elements so that $w(e_1) \ge w(e_2) \ge \cdots \ge w(e_n)$.
  2. Initialize $I \leftarrow \emptyset$.
  3. For $i = 1, 2, \ldots, n$: if $I \cup \{e_i\} \in \mathcal{I}$, set $I \leftarrow I \cup \{e_i\}$.

Output $I$. The cost is $O(n \log n)$ for the sort plus $n$ independence tests. The classic application is Kruskal's minimum-spanning-tree algorithm: take the graphic matroid, weights = edge weights, greedy = Kruskal.

Rado–Edmonds: matroids are exactly where greedy works

The deepest theorem of elementary matroid theory:

Theorem (Rado 1957, Edmonds 1971). Let $\mathcal{I}$ be a non-empty hereditary family on $E$. The greedy algorithm finds a maximum-weight independent set for every non-negative weight function $w$ if and only if $\mathcal{I}$ satisfies the exchange axiom (I3) — i.e. iff $(E, \mathcal{I})$ is a matroid.

One direction ($\Leftarrow$): suppose $\mathcal{I}$ is a matroid and the greedy returns $I = \{e_{i_1}, \ldots, e_{i_k}\}$ with $w(e_{i_1}) \ge \cdots \ge w(e_{i_k})$. Suppose for contradiction some $J \in \mathcal{I}$ has strictly greater total weight, with $J = \{e_{j_1}, \ldots, e_{j_\ell}\}$ in decreasing order. Both $I$ and $J$ are independent and $|I|, |J| \le r(M)$. Apply (I3) to extend $I_{ w(I)$.

The reverse direction ($\Rightarrow$) is more striking: hereditary plus "greedy is always optimal" forces the exchange axiom. So matroids are characterized by the optimality of greedy. Whenever a combinatorial-optimization problem on a hereditary family is solved by sort-and-take, the underlying structure is a matroid; whenever the greedy fails on some instance, you have built a non-matroid.

Matroid intersection (sketch)

Given two matroids $M_1, M_2$ on the same ground set, finding a maximum-cardinality common independent set $I \in \mathcal{I}_1 \cap \mathcal{I}_2$ is solvable in polynomial time (Edmonds 1970). The intersection captures bipartite matching ($M_1, M_2$ = partition matroids on the two sides), arborescences (graphic + partition), and many more. Three matroids' intersection is NP-hard; two is the boundary of tractability.

7. The Tutte polynomial

The deepest invariant of a matroid is a two-variable polynomial whose specializations enumerate spanning trees, count colorings, compute reliability, and read out the partition function of statistical-mechanics models. Define the Tutte polynomial via corank–nullity: $$T_M(x,y) \;=\; \sum_{S \subseteq E} (x-1)^{r(E) - r(S)} \, (y-1)^{|S| - r(S)}.$$ The exponent $r(E) - r(S)$ is the corank of $S$ (how far short of full rank); $|S| - r(S)$ is the nullity (how many cycles $S$ contains). Despite the strange shifts, $T_M(x,y) \in \mathbb{Z}_{\ge 0}[x,y]$ — every coefficient is a non-negative integer (Tutte 1947).

The deletion–contraction recursion

For an element $e \in E$ that is neither a loop nor a coloop: $$T_M(x,y) \;=\; T_{M \setminus e}(x,y) + T_{M / e}(x,y).$$ Here $M \setminus e$ is the matroid deletion (remove $e$, restrict $\mathcal{I}$) and $M/e$ is the contraction (treat $e$ as committed; rank drops by $1$). Loops and coloops produce boundary cases: $T_M = y \cdot T_{M \setminus e}$ if $e$ is a loop, and $T_M = x \cdot T_{M / e}$ if $e$ is a coloop. The recursion plus the boundary identities computes $T_M$ from scratch in polynomial-time-per-leaf but exponentially-many-leaves recursion: deciding whether $T_M(x,y) = 0$ at a generic point is $\#P$-hard.
